## 依赖

- 🚀 vue3.2 + vite + typescript + pinia + axios + vant
- 💪 使用 vue3.2 `<script setup>` 语法糖
- 💪 使用 TypeScript
- 🍭 支持 jsx 组件写法
- 🍭 整合 vant ui、less
- 🍭 使用 viewport 移动端方案
- 🍭 使用 pinia 替代 vuex，更简单、更高效
- 🍭 使用 `网易云音乐` 热门歌曲接口为数据源

## 启动、部署

Using `yarn` to :

````bash
  # 安装依赖
  yarn

  # 本地启动
  yarn dev

  # 本地打包
  yarn build
````
